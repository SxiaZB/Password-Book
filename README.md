### 这是什么？干什么用的？

    这是一个密码本小程序，用来记录密码的！

### 为什么要写？

    用来替换掉我钱包中的那张记录了几十个密码的纸:)

### 记录的密码安全吗？

    1. 这是个开源软件，意味着：这几乎不可能是一个窃取你密码的恶意程序。

    更重要的一点：
        此小程序不会与任何外界服务器发送数据，可以通过这个方法检验： @附上截图
        记录的密码数据只存储在手机的当前登录用户上。如果此手机换了微信登录用户，也无法访问到其他用户创建的数据（由微信来保证）

    2. 我也只是用来记录一下重要程度不是很高的密码
    
    so. 从以上几方面来讲，是安全的。
